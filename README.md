# Домашнее задание к работе 4
## Условие задачи
16. Посадка семян

Садовод-робот сажает семена в лунки. Лунка считается "идеальной" для посадки, если все три измеренных параметра кислотности почвы (A, B, C) кратны трем. Запишите условие для "идеальной" лунки.
## 1. Алгоритм и блок-схема
### Алгоритм
1. **Начало**
2. ввод значений :
   - `A` = первое число.
   - `B` = второе число.
   - `C` = третье число.
3.  Проверка кратности:
   ((A % 3 == 0) && (B % 3 == 0) && (C % 3 == 0))
4. Выводим результаты расчетов
5. **Конец**

## 2. Реализация программы

#define _CRT_SECURE_NO_WARNINGS
#include <locale.h>
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <math.h>

void main()
{
	setlocale(LC_CTYPE, "RUS");
	int A = 3;
	int B = 6;
	int C = 94;
	if ((A % 3 == 0) && (B % 3 == 0) && (C % 3 == 0)) {
		printf("Идеальная лунка");
	}
	else {
		printf("Плохая лунка");
	}

}
